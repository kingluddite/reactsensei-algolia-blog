## Keyboard Shortcuts 
| Command | Description |
| ------- | -------- |
| `zM` | close everything | 
| `zR` | open everything | 
| `za` | toggle state of the current fold | 
| `zj` | jump down to next fold | 
| `zk` | jump up to previous fold | 
| `zk` | jump up to previous fold | 
| `zO` | Open all of the nested folds | 
| `zc` | Close all of the nested folds
